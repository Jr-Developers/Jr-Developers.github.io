Spring Initializr를 사용한 Spring Boot 프로젝트 생성

by 주니어 개발자

December 8, 2020
in Spring-boot

Spring에서는 Spring Boot 프로젝트의 생성을 위해 Dependency, Build Tool, 언어등을 쉽게 선택하여 프로젝트를 생성할 수 있는 사이트를 제공한다.

또한 Intellij에서도 File > New > Project...에서도 Spring Initializr를 통해 생성이 가능하다.

Spring Initializr를 통해 생성한 프로젝트의 간단한 테스트 결과 정상적으로 Spring Boot 프로젝트가 동작하는 것을 확인하였다.
