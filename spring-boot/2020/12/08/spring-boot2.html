여러 Spring Boot 서버 H2 접속 설정

by 주니어 개발자

December 8, 2020
in Spring-boot

여러 Spring Boot 서버에서 H2 서버에 접속해야 할 경우 다음과 같이 설정한다.
우선 하나의 서버에서 다음과 같은 @Configuration 설정을 추가하며 해당 서버를 A서버라고 지칭하겠다.

A서버의 H2에 다른 Spring Boot 서버도 접근 가능하다록 TCP 포트를 개방하는 설정이다.
다음은 A서버 application.yml 설정으로 초기 테이블 생성과 데이터 생성을 위한 설정을 하였다.

  • application.yml - A서버

A서버 프로젝트의 자세한 소스는 여기를 확인하며 A서버가 아닌 다른 접근 하는 서버를 B서버라고 지칭하겠다.
B서버에서는 단순히 A서버의 H2 서버의 데이터를 조회 할 것이다. 다음은 B서버의 application.yml이다.

  • application.yml - B서버

B서버 프로젝트의 자세한 소스는 여기를 확인하며 A서버에서 기동 중이고 포트를 개방한 H2 로 접속한다.

다음은 두 서버의 Spring Boot 메인으로 테스트를 위한 간단한 조회 결과를 응답한다.

  • SpringBootMain
  • 확인 결과

A서버에서 H2가 기동되므로 A서버 기동 이후 B서버를 기동하여 두 서버 모두 같은 조회 결과를 응답하는 것을 확인하였다.
